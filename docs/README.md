# 📚 Gestion Bibliothèque Pro

Application web moderne de gestion de bibliothèque développée en HTML5, CSS3, JavaScript Vanilla et XML.

Ce projet a été réalisé dans le cadre du module :

JavaScript Avancé

---
 
# 🎯 Objectif du Projet

L’objectif principal est de développer une application web interactive permettant :

- le chargement dynamique de données XML
- l’affichage des livres dans un tableau
- l’ajout de nouveaux livres
- la modification des informations
- la suppression des livres
- la recherche dynamique par titre
- l’affichage détaillé via une modal

Le projet met en pratique :

- la manipulation du DOM
- la gestion des événements
- le CRUD en JavaScript
- le chargement XML avec XMLHttpRequest
- la logique applicative front-end

---

# 🛠️ Technologies Utilisées

- HTML5
- CSS3
- JavaScript Vanilla
- XML
- DOM Manipulation
- XMLHttpRequest
- VS Code
- Live Server
